# Privacy Policy for WhatsApp Integration

Last updated: June 2026

## 1. Data Collection
This application is currently in development. We only process data required for core app functionality and webhook testing. We do not collect or store personal user data for production use.

## 2. Data Usage
Data received via Meta APIs is used solely to execute the requested actions within the application and is not shared with third parties.

## 3. Data Deletion
Users can request data deletion by contacting the developer directly through the app support channels.
